# MVP Scope

## 포함 범위

- 프리랜서·플랫폼 노동자 대상 개인신용대출 1종
- 개인정보를 포함하지 않는 합성 금융 데이터
- Loan Decision Agent
- RippleGuard Agent
- Evidence & Control Agent
- 결정론적 Governance Service와 정책 평가
- 관리자의 Agent 활성·Shadow·비활성 통제
- 실행 Trace와 Replay

## 제외 범위

- 실제 대출금 지급과 실제 금융기관 연동
- 복수 금융기관 또는 복수 대출상품
- 금리 최적화와 복잡한 법률 자문
- Agent Builder와 실시간 Prompt 편집
- Kubernetes 배포

제외 항목은 MVP에서 검증하려는 2차 위험 통제와 직접 관련되지 않거나 운영·규제 범위를 크게 확장한다. 후속 Phase 편입은 별도 결정과 ADR이 필요하다.
